### 기능 소개 

- 현재 기록하기
- 미래 목표 추가하기 
- 미래 목표 설정 및 현재가 되었을 때 저장하기
- CES 지수에 따른 통계 자료 생성(그래프) 
- CES 지수를 통한 AI 요약 보고서 생성
- 과거 탭에서 저장된 기억들 보기
